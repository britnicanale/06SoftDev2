Numpy

What is Numpy?

  • Numpy is a free, open-source Python Library.
  • Can be seen as an alternative to MatLab.
  • It provides a single-typed array that is capable of vector and matrix calculations.

Why not use Python arrays?

  • Numpy arrays are more compact. A 3D Numpy Array takes up 4MB of data; regular arrays take up 20MB.
  • No need to check data type.
  • Allows for vector and matrix calculations.
  • Faster access.
  • Better for large amounts of data.

Array Anatomy



How to create an array

import Numpy as np

A = np.array([[3, 4], [6, 8], [7, 2]])

print(A.shape) # --> (3, 2)

print(A.dtype) # default for ints is int64

Matrix calculations

import Numpy as np

A = np.eye(dimension) # returns a dimension x dimension identity matrix

np.dot(A, B) # returns dot product of two matrices A & B

np.sum(A) # returns sum of elements in matrix A

np.sum(A, axis = a) # returns sum of elements in axis a of matrix A

# Element-Wise Computations: uses normal operators; Ex:
A + B # adds matrices A & B
np.add(A, B) # adds matrices A & B

A.dot(B) # returns dot product of two matrices A & B

# Transposing: switch rows and columns of matrix
A = np.array([1, 2], [3, 4])
print(A.T) # returns [[1, 3], [2, 4]]

# Boolean Indexing:
A = np.array([[1, 3], [2, 4]])
Bool_idx = A > 2
print(A[bool_idx]) # returns [[false, true],[false, true]]


Scipy

What is Scipy?

  • Scipy is a free, python-based, open-source software.
  • Contains implementations of numerical calculations.
  • Uses Numpyâ€™s ndarray to perform computations.
  • Works through wrappers around scientific algorithms written in C from NetLib.
  • Can be used with MatPlotLib for 3D Plotting.

How can we use Scipy with Numpy?

Some ~mathy things~ that can be done using Scipy (with their submodule names. Scipy has been imported as import Scipy):
  • Linear Algebra --> Scipy.linalg
  • Optimization & Root-finding --> Scipy.optimize
  • Statistics --> Scipy.stats
  • Sparse Matrices --> Scipy.sparse
  • Spatial data structures --> Scipy.spatial
  • Integration and differentiation --> Scipy.integrate
Some useful functions using Linear Algebra

import Numpy as np
from Scipy import linalg

A = np.array([[1, 3], [2, 4]])
linalg.det(A) # returns determinant of array A

linalg.inv(A) # returns inverse of array A
